# Medical Research Assistant

AI-powered research tool using RAG (Retrieval-Augmented Generation).

## Features
- Upload PDF research papers
- Ask questions in natural language
- Get AI-synthesized answers with citations
- Powered by TF-IDF retrieval and Groq AI

## How to use
1. Get a free API key from [console.groq.com](https://console.groq.com)
2. Upload your research papers (PDF format)
3. Ask your question
4. Receive evidence-based answers with source citations
